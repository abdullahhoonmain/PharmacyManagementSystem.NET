Pharmacy Management System.NET is a web based AI powered Pharmacy Managing App developed using ASP.NET Core MVC and Entity Framework. It is designed to manage daily based basic operations including Medicines, Customers, Suppliers, Bills, Bill Details and Employees.
It has 3 employee levels 1. Admin 2. Pharmacist 3. Cashier.
To download and use it. You should have NET 8.0 and Microsoft.EntityFrameworks and Microsoft.EntityFramework.Core and Microsoft.Sql. All 8.0 verions
Run the migrations, Update the ConnectionString in program.cs and appSetting.js. And you are good to go.
